Telnet server:
查看telnet server是否安装
rpm -qa | grep telnet-server
如果没有，则安装：
yum install telnet-server -y
开启telnet服务：
chkconfig telnet on
启动telnet服务：
service xinetd reload

Center OS 7:
systemctl start telnet.socket 
systemctl enable telnet.socket  开机启动
systemctl status telnet.socket
